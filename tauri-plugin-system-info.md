# Tauri Plugin System Info

> A tauri plugin written in Rust and TypeScript to provide system information to a tauri desktop app.

Added to [awesome-tauri](https://github.com/tauri-apps/awesome-tauri)

## Features

- CPU
- Network
- Process
- Memory
- Hostname
- Kernel Version
- OS Version
- Battery
